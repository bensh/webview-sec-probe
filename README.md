Quick dirty PoC [here](https://bensh.github.io/webview-sec-probe) to test Webview issues on mobile apps.

Useful redirects:
[Google](https://google.com)


## Usage
- Grab a webhook URL from [webhook.site](https://webhook.site) and paste it into the top box. Alternatively use a Burp Collaborator URL.
- Add your native JS handlers
- Add file paths to test locally
- Click Run all test
- Check output of your hooking URL
